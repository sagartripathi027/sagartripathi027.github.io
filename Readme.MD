## 🚀 Run Locally

**Option 1 — Just double-click:**
```
Open index.html in any browser
```

**Option 2 — VS Code Live Server:**
1. Install the **Live Server** extension in VS Code
2. Right-click `index.html` → **Open with Live Server**
3. Visit `http://127.0.0.1:5500/index.html`

## 🌐 Deploy to GitHub Pages

```bash
git init
git add .
git commit -m "initial commit"
git branch -M main
git remote add origin https://github.com/sagartripathi027/sagartripathi027.github.io.git
git push -u origin main
```

Then go to: **Settings → Pages → Source: main → Save**

Live at: `https://sagartripathi027.github.io` 🎉
